# Robot Gladiators

## Purpose
Use JS code to run a turn based fighting game 

## Features
* Customized player names using window.prompt
* Randomized attack/health values
* Shop feature that allows you to refill health or upgrade attack damage
* Randomized fight() function combat turns between player and enemy robots
* Highscore is saved based on money value within playerInfo object
